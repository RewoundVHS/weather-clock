## Raspberry Pi GPIO
```
| Display | Raspberry Pi GPIO |
|:-------:|:-----------------:|
| VCC     | 3.3V              |
| GND     | GND               |
| DIN     | 19                |
| CLK     | 23                |
| CS      | 24                |
| DC      | 22                |
| RST     | 11                |
| BUSY    | 18                |
```
